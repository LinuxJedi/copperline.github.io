Projects using Copperline

Things other people have built with or around Copperline. These are independent projects with their own maintainers, so issues and pull requests belong with them. Built something yourself? Open an issue on the site repository and I'll add it here.

  • by Retro32

    Retro32 BBS

    Retro gaming store Retro32 host the browser build of Copperline on their site as the front door to their BBS. The page uses the serial/internet bridging in the browser build to wire the emulated Amiga's serial port through to the BBS, so one press of Connect and you're dialling in from a virtual Amiga right there in the page - nothing to install.

  • by Simon Dick

    copperline-docker

    The browser build of Copperline, self-hosted: a single Docker container that compiles the WebAssembly bundle and serves it with nginx. Pre-built multi-arch images (amd64 / arm64) are on GHCR - docker run, open localhost:8080, and it boots the bundled AROS ROM. Mount a host directory into the container and it serves your own Kickstart and disk images to the emulator by URL.
